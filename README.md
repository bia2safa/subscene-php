# rifalpg/subcene-php

Library For searching, downloading, and extract subtitle from [https://subscene.com](https://subscene.com).
This is not officialy from subscene. I made just for fun only!

# Installation

You can install the package via composer:

```php
composer require rifalpg/subscene-php
```

### Requirements
- curl extension enabled
- (Optional) ZipArchive Installed (http://pecl.php.net/package/zip/docs)
